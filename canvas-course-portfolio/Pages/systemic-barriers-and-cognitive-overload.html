Systemic Barriers & Cognitive Overload

Key Systemic Barriers

Now that you've identified barriers firsthand, let's explore the psychological mechanisms that explain why these design choices are so problematic for neurodivergent learners: 

Cognitive Overload: Densely packed text, complex navigation, or excessive user workload, like fragmented platform integrations, maxes out working memory, triggering ND symptoms like task paralysis (Le Cunff et al., 2024; Tcherdakoff et al., 2025).

System Failures: Inconsistent platform integration, unclear headings, and frustrating login processes create a landscape of obstacles on the digital campus (Tcherdakoff et al., 2025).

The Cognitive Mechanism: Cognitive Load Theory (CLT)

Cognitive Load Theory (CLT) provides the foundational psychological framework for why inaccessible design is detrimental to learning, particularly for ND students.

CLT is based on the assumption that working memory capacity is limited, while long-term memory is vast (Hollender, 2010). The basic assertion is that instructional design must account for working memory limitations to prevent overload and deterioration in learning.

The Three Types of Cognitive Load

The total cognitive load must remain within an individual's capacity. Instructional design seeks to manage the three additive types of load (Hollender, 2010; Le Cunff et al., 2024):

 

Table 1: Overview of cognitive load types and their significance in designing accessible, inclusive learning experiences.
Cognitive Load Type Definition & Source Relevance to Inclusive Design
Intrinsic Load (ICL) The inherent difficulty of the information itself. This depends on the element's interactivity with the topic and the learner's existing expertise (Hollender, 2010). Focuses on simplifying complex content (e.g., presenting isolated elements first). NT students often report this as their primary challenge (Le Cunff et al., 2024).
Extraneous Load (ECL) Load caused by inappropriate instructional design or irrelevant activities, such as poorly designed interfaces or having to integrate information from separate, fragmented sources (Hollender, 2010). Crucial for Neurodivergence: Accessibility-related difficulties (e.g., a confusing LMS interface, poor navigation) are forms of extraneous load that consume working memory, leading to cognitive overload. ND students often report ECL as their primary challenge (Le Cunff et al., 2024).
Germane Load (GCL) The beneficial load that results from the learner's active effort to construct and automate schemata (knowledge structures) in long-term memory (Hollender, 2010; Le Cunff et al., 2024). The Goal: Effective instructional design minimizes Extraneous Load to maximize Germane Load, freeing up working memory for true learning processes (Hollender, 2010).
